Reptiles - symfony2 app
=======================

How to run the application?

Clone the repo:

    $ git clone https://github.com/pro-vagrant/reptiles-symfony2-app.git
    $ cd reptiles-symfony2-app

Boot VM:

    $ vagrant up
    $ vagrant ssh
    $ cd /vagrant

Install the dependencies:

    $ composer install

Finally start your web browser and visit:

    http://127.0.0.1:8880
